# Ghost Reader — macOS 透明悬浮电子阅读器设计文档

## 概述

Ghost Reader 是一个 macOS 桌面电子阅读器应用，核心特点是透明悬浮窗设计，可以覆盖在 IDE 等工作窗口之上，实现"摸鱼式阅读"体验。支持 EPUB 和 TXT 格式。

## 技术栈

- **运行时：** Electron
- **前端：** React + TypeScript
- **构建：** Vite
- **EPUB 解析：** epub.js
- **配置存储：** electron-store
- **包管理：** Bun

## 架构

### 进程模型

```
Main Process（主进程）
├── 窗口管理（透明、置顶、位置记忆）
├── 文件系统（打开/扫描书籍）
├── 快捷键注册
└── 配置存储（electron-store）

Renderer Process（渲染进程）
├── 阅读器视图（EPUB / TXT 渲染）
├── 书架视图（书籍列表管理）
├── 透明度/渐隐控制
└── 滚动/翻页逻辑
```

### 两个窗口模式

1. **书架模式** — 普通窗口，用于选书、导入书籍、管理设置
2. **阅读模式** — 透明无边框悬浮窗，纯阅读

## 核心功能：透明悬浮窗 & 渐隐机制

### 窗口属性

```typescript
const readerWindow = new BrowserWindow({
  transparent: true,       // 窗口背景透明
  frame: false,            // 无标题栏/边框
  alwaysOnTop: true,       // 始终置顶
  hasShadow: false,        // 无阴影，更隐蔽
  skipTaskbar: true,       // 不在 Dock 显示
  visibleOnAllWorkspaces: true, // 多桌面可见
});
```

### 两态模型

应用在阅读模式下有两个状态：

#### 隐匿态（默认）

- 透明度：0.10（几乎全透明）
- 鼠标行为：
  - **单击** → 穿透到下方窗口（IDE）
  - **双击** → 激活，切换到阅读态（300ms 渐现动画）
  - **鼠标移动** → 无反应，不会意外激活
  - **滚轮** → 穿透到下方窗口
- 实现：`setIgnoreMouseEvents(true, {forward: true})`

#### 阅读态

- 透明度：0.85（半透明深色背景）
- 鼠标行为：
  - **滚轮/触控板** → 滚动书的内容
  - **↑/↓ 键** → 滚动一行
  - **⌘+↑/⌘+↓** → 翻一页
  - **鼠标移出 1 秒** → 渐隐回隐匿态（300ms 动画）
- 实现：`setIgnoreMouseEvents(false)`

#### 状态流转

```
隐匿态 --[双击]--> 阅读态
阅读态 --[鼠标移出 1s]--> 隐匿态（300ms 渐隐）
```

### 双击检测

隐匿态下通过 `forward: true` 选项仍可接收鼠标事件通知，监听 dblclick 事件（250ms 窗口判定）来激活阅读态。

## 文件格式支持

### EPUB

- 使用 epub.js 解析和渲染
- EPUB 本质是 HTML/CSS，直接在 WebView 中渲染
- 支持章节导航、阅读进度记忆

### TXT

- 纯文本加载，自动分段
- 应用阅读样式（字体、行间距、段间距）
- 支持编码检测（UTF-8、GBK 等）

## 窗口控制

### 位置与大小

- 自由拖拽到任意位置
- 支持右侧吸附（靠近屏幕右侧时自动对齐）
- 位置和大小持久化，下次打开恢复上次状态
- 通过 electron-store 存储窗口 bounds

### 拖拽实现

阅读态下，窗口顶部保留一个细窄的拖拽区域（CSS `-webkit-app-region: drag`），其余区域正常交互。

## 书架视图

- 列出已导入的书籍（封面 + 书名 + 作者 + 阅读进度）
- 支持拖入文件导入
- 支持从文件选择器导入
- 点击书籍进入阅读模式

## 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 阅读态透明度 | 0.85 | 阅读时的背景不透明度 |
| 隐匿态透明度 | 0.10 | 隐匿时的背景不透明度 |
| 渐隐延迟 | 1000ms | 鼠标移出后多久开始渐隐 |
| 渐隐动画时长 | 300ms | 渐隐/渐现动画时长 |
| 字体大小 | 16px | 阅读字体大小 |
| 行间距 | 1.8 | 行间距倍数 |
| 快捷键 | 可配置 | ↑/↓ 滚动一行，⌘+↑/↓ 翻页 |

## 第一版范围

### 包含

- Electron 透明无边框悬浮窗
- 双击激活 / 鼠标移出渐隐的两态模型
- EPUB 解析与渲染（epub.js）
- TXT 加载与渲染
- 触控板滚动 + 键盘翻页
- 窗口自由拖拽 + 右侧吸附 + 位置记忆
- 书架视图（导入、列表、进度）
- 阅读进度自动保存
- 基本设置（透明度、字体、快捷键）

### 不包含

- 书签/批注
- 搜索功能
- 多主题（暂用深色主题）
- 自动更新
- 跨平台支持
