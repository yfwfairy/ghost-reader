# Ghost Reader v1 Release Manual Checklist

> 适用场景：`v1` 发布前手工验收 / smoke test  
> 版本：`0.1.0`  
> 创建日期：2026-04-14

---

## 0. 验收准备

- [ ] 使用一份干净的用户数据环境启动应用
- [ ] 准备至少 3 个测试文件：
  - [ ] 1 本普通 TXT
  - [ ] 1 本带封面的 EPUB
  - [ ] 1 本无封面或 TOC 较简单的 EPUB
- [ ] 明确本轮验收设备与系统版本（建议记录在备注区）

---

## 1. 启动与窗口

- [ ] 启动应用，只创建一个主窗口
- [ ] 首页正常进入书架，而不是直接掉进阅读器
- [ ] 首页窗口可以拉宽和缩窄，且不会缩到不可用
- [ ] 打开阅读器后窗口最小尺寸切换正常，界面不挤坏
- [ ] 标题栏区域可拖动窗口
- [ ] `Pin` / `置顶` 按钮可用，置顶状态切换无异常

---

## 2. 书库主流程

- [ ] 首页默认进入 `Library`
- [ ] 点击 `Add to Library` / 添加书籍可打开系统文件选择框
- [ ] 可导入 TXT 文件
- [ ] 可导入 EPUB 文件
- [ ] 导入后书籍立即出现在书库中
- [ ] 搜索可按标题或作者过滤结果
- [ ] 删除书籍后，该书从 `Library` 消失
- [ ] 删除已读书籍后，该书也从 `Recent` 消失
- [ ] 空书库状态下仍然能看到导入入口

---

## 3. 最近阅读主流程

- [ ] 点击 `Recent` 可切换到最近阅读视图，左侧壳层保持稳定
- [ ] 未产生阅读进度前，`Recent` 显示空状态而不是假数据
- [ ] 空状态下点击“浏览书库”能回到 `Library`
- [ ] 打开一本书并产生阅读进度后，该书进入 `Recent`
- [ ] `Recent` 只显示有阅读进度的书
- [ ] 多本书都有进度时，排序符合“最近阅读在前”
- [ ] 点击 `Recent` 卡片可进入阅读器

---

## 4. TXT 阅读器

- [ ] 从 `Library` 打开 TXT，内容按段落正常渲染
- [ ] 中文 TXT 显示正常，无明显乱码
- [ ] 滚动阅读稳定，无闪屏或跳动
- [ ] 返回书架后再次打开同一本 TXT，可恢复到上次位置附近
- [ ] 从阅读器返回后，能稳定回到主界面壳层

---

## 5. EPUB 阅读器

- [ ] 从 `Library` 打开 EPUB，正文能正常渲染
- [ ] EPUB 打开后无明显闪屏
- [ ] 带封面的 EPUB 在书库 / 最近阅读中能显示封面
- [ ] 无封面的 EPUB 有合理的占位封面
- [ ] 返回书架后再次打开同一本 EPUB，可恢复到上次位置附近
- [ ] 目录抽屉可打开，章节点击跳转正常
- [ ] 上一章 / 下一章流程正常，不会把阅读器带坏

---

## 6. 阅读器界面与交互

- [ ] 阅读器顶部返回操作可用
- [ ] 阅读器标题、作者、进度等基础信息显示正常
- [ ] 底部导航可打开 `章节` / `外观` 抽屉
- [ ] 沉浸模式可进入，也可正常退出
- [ ] 沉浸模式下内容仍可阅读，滚动不被顶部拖拽区干扰
- [ ] `F11` 切换沉浸模式无异常

---

## 7. 设置、主题与语言

- [ ] 侧边栏 `设置` 可打开设置面板
- [ ] 设置面板可通过关闭按钮或点击遮罩关闭
- [ ] 深色主题可切换并即时生效
- [ ] 浅色主题可切换并即时生效
- [ ] “跟随系统”开启后，系统深浅切换能带动应用切换
- [ ] 语言可切到 `English`
- [ ] 语言可切到 `简体中文`
- [ ] 语言可切到 `繁體中文`
- [ ] 切换语言后，首页 / 设置 / 阅读器核心文案同步更新

---

## 8. 持久化与重启回归

- [ ] 关闭应用再启动，书库列表仍在
- [ ] 关闭应用再启动，最近阅读仍在
- [ ] 关闭应用再启动，阅读进度仍在
- [ ] 关闭应用再启动，主题设置仍在
- [ ] 关闭应用再启动，语言设置仍在
- [ ] 关闭应用再启动，置顶状态仍在

---

## 9. 边界与回归

- [ ] 重复导入同一本书，不产生重复记录
- [ ] 超长书名不会把卡片布局撑坏
- [ ] unknown 作者在卡片中不会显示脏文案
- [ ] 打开已不存在的文件时，应用不会直接崩溃
- [ ] 连续执行“导入 -> 打开 -> 返回 -> 删除 -> 重启”后，主流程仍然正常

---

## 10. 发布结论

- [ ] 本轮没有阻塞发布的崩溃、卡死、白屏、数据丢失问题
- [ ] 本轮没有阻塞发布的核心流程断裂问题
- [ ] 若存在已知问题，已经单独记录为 release note / 已知限制
- [ ] 可以产出可安装版本给自己或首批测试用户试用
